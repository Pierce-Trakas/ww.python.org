Exponential Functions

Exponential Growth and Decay

Exponential Functions have a structure of y = a(r)x, with a representing the initial value of the function (i.e. the y value of the function while intersecting the y-axis), and r representing the rate of growth or decay. If r > 1, then the value of the function will grow as the x value increases. The opposite will occur if r is between 1 and 0. The gradual increase or decrease of exponential functions makes them a handy way to graph out compound interest.

Exponential Graphs and Transformations

Every exponential function has Asymptotes, which are lines that the function approaches, but does not cross. Depending on how you transform the function, the asymptote's location will change, as well as the function's domain and range.

Logarithms

To find out how many times a number has been multiplied (the x value) by itself in an exponential function when given the equation, you can use Logarithms on your calculator. Logarithms are structured as x = logr(y), consisting of a base (r), exponent(x), and argument (y). An example is log10(100), which calculates the total amount of times that 100 can be divided by 10 while still being an integer, thus outputting 2 (since 102 is 100). Another example is log2(64), which gives an answer of 6, as 26 is 64.
